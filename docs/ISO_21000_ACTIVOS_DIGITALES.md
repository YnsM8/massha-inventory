# Gestión de Activos Intangibles - ISO 21000 (Adaptado)
## MASSHA'S CATERING

### 1. Principio
Esta norma, generalmente enfocada en derechos multimedia, se adapta aquí para la protección de los **activos de información críticos** del negocio de catering.

### 2. Activos Protegidos por el Sistema
El sistema "MASSHA'S INVENTORY" actúa como repositorio seguro para:

* **Base de Datos de Proveedores:** Información comercial sensible y acuerdos de precios.
* **Historial de Eventos:** Datos históricos que constituyen el "know-how" operativo de la empresa.

### 3. Medidas de Protección
* La propiedad intelectual (código fuente) está protegida mediante control de versiones privado.
* El acceso a los datos comerciales está restringido exclusivamente a personal autorizado mediante autenticación segura.
